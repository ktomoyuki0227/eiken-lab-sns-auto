# Claude for Chrome 用プロンプト集

作成日：2026-04-06
更新日：2026-04-07

各プロンプトをそのまま Claude for Chrome のサイドパネルに貼り付けて使う。
1ステップずつ完了を確認してから次へ進むこと。

| ステップ | 担当 | 実施アカウント |
|---------|------|--------------|
| STEP 1 | 友幸 | Dify アカウント |
| STEP 2 | 長畑さん | 長畑さんの Google アカウント |
| STEP 3 | 長畑さん | 長畑さんの Google アカウント |
| STEP 4 | 長畑さん | 長畑さんの Google アカウント |
| STEP 5 | 長畑さん | 長畑さんの Google アカウント |

STEP 2〜5 のプロンプトは長畑さんに送付して、長畑さんのブラウザで実行してもらう。

## STEP 1：Dify ワークスペース確認

```
以下の手順で Dify のワークスペースを確認してください。

1. https://cloud.dify.ai を開いてください。
2. 左上に表示されているワークスペース名を教えてください。
3. 左サイドバーの「Studio」を開き、「Create App」ボタンが表示されていることを確認してください。

確認できたことをすべて報告してください。
```

## STEP 2：Google Cloud プロジェクト作成【長畑さん担当】

```
以下の手順で Google Cloud のプロジェクトを作成してください。
長畑さんのGoogle アカウントでログインした状態で実行してください。

1. https://console.cloud.google.com を開いてください。
2. 画面上部のプロジェクト選択ドロップダウンをクリックしてください。
3. 「新しいプロジェクト」をクリックしてください。
4. プロジェクト名に「eiken-lab-sns」と入力してください。
5. 「作成」をクリックしてください。
6. 作成が完了したら、プロジェクト ID を画面から読み取って教えてください。
```

## STEP 3：Google Sheets API 有効化【長畑さん担当】

```
（プロジェクト「eiken-lab-sns」が選択されていることを確認してから実行してください）

以下の手順で Google Sheets API を有効化してください。

1. https://console.cloud.google.com を開いてください。
2. 左メニューから「APIとサービス」→「ライブラリ」を開いてください。
3. 検索ボックスに「Google Sheets API」と入力してください。
4. 検索結果から「Google Sheets API」をクリックしてください。
5. 「有効にする」ボタンをクリックしてください。
6. 有効化が完了したことを確認して報告してください。
```

## STEP 4：サービスアカウント作成・JSON キーのダウンロード【長畑さん担当】

```
以下の手順でサービスアカウントを作成し、JSON キーをダウンロードしてください。

1. https://console.cloud.google.com を開いてください。
2. 左メニューから「APIとサービス」→「認証情報」を開いてください。
3. 上部の「認証情報を作成」→「サービスアカウント」をクリックしてください。
4. サービスアカウント名に「dify-sheets-writer」と入力してください。
5. 「作成して続行」をクリックしてください。
6. ロールの選択で「編集者」を選んでください。
7. 「完了」をクリックしてください。
8. 認証情報ページに戻り、作成したサービスアカウント「dify-sheets-writer」の行をクリックしてください。
9. 「キー」タブを開いてください。
10. 「鍵を追加」→「新しい鍵を作成」→「JSON」を選択して「作成」をクリックしてください。
11. JSON ファイルがダウンロードされます。ダウンロードされたファイル名を教えてください。
12. ダウンロードした JSON ファイルの中から「client_email」の値を読み取って教えてください。

※ JSON ファイルは Git に絶対に入れないでください。
※ JSON ファイルは友幸に安全な方法で送付してください（メール添付 等）。
```

## STEP 5：Google スプレッドシート作成・列設定・共有【長畑さん担当】

```
以下の手順で Google スプレッドシートを作成してください。

1. 以下の URL を開いてください（長畑さん提供の Google Drive フォルダ）：
   https://drive.google.com/drive/folders/1NCD5b8Tg7YPQMCm4tEC9xj26Onx8i0uK
2. フォルダ内で「新規」→「Google スプレッドシート」→「空白のスプレッドシート」を作成してください。
3. スプレッドシートのタイトルを「英検ラボ SNS コンテンツ候補」に変更してください。

【シート1（Instagram）のヘッダー設定】
4. シート名のタブをダブルクリックして「Instagram」に変更してください。
5. 1行目に以下のヘッダーを A 列から順番に入力してください：
   A: 生成日　B: カテゴリ　C: タイトル　D: 台本_Hook(0-3秒)　E: 台本_Agitate(4-11秒)　F: 台本_Solution(12-18秒)　G: キャプション　H: ハッシュタグ　I: 採用フラグ　J: 備考

【シート2（note）の作成】
6. 画面下部の「＋」をクリックして新しいシートを追加してください。
7. シート名を「note」に変更してください。
8. 1行目に以下のヘッダーを A 列から順番に入力してください：
   A: 生成日　B: ターゲット　C: カテゴリ　D: タイトル　E: リード文　F: 本文　G: まとめ　H: 採用フラグ　I: 備考

【サービスアカウントと共有】
9. 右上の「共有」ボタンをクリックしてください。
10. メールアドレス入力欄に [STEP 4 で取得した client_email を入れる] を入力してください。
11. 権限を「編集者」に設定して「送信」してください。

【スプレッドシート ID の取得】
12. URL バーのアドレスを確認してください。
    https://docs.google.com/spreadsheets/d/【ここの文字列】/edit
    この部分の文字列（スプレッドシート ID）を私に教えてください。
```

## STEP 6：Dify ワークフロー新規作成

```
以下の手順で Dify に新しいワークフローアプリを作成してください。

1. https://cloud.dify.ai を開いてください。
2. 左サイドバーの「Studio」をクリックしてください。
3. 「Create App」ボタンをクリックしてください。
4. 「Workflow」を選択してください。
5. アプリ名に「eiken-lab-daily-generator」と入力してください。
6. 説明欄に「英検ラボ SNS コンテンツ毎日自動生成」と入力してください。
7. 「Create」をクリックしてください。
8. ワークフロー編集画面が開いたら、画面に表示されているノード一覧を教えてください。
```

## 使い方のヒント

- 各プロンプトは1つずつ実行し、Claude が完了を報告してから次へ進む
- STEP 4 の `client_email` と STEP 5 の `spreadsheetId` は必ずメモしておく
- JSON キーのダウンロード後はプロジェクトフォルダには置かない（別フォルダに保管）
